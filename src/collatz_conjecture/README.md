## 📝 문제 요약

- 자연수 n이 주어지면, n이 1이 될때까지 다음 작업을 반복하여 몇 번만에 1이 되는지 반환하는 문제
  - 1-1. 입력된 수가 짝수라면 2로 나눕니다.
  - 1-2. 입력된 수가 홀수라면 3을 곱하고 1을 더합니다.
  - 2. 결과로 나온 수에 같은 작업을 1이 될 때까지 반복합니다.

---

## 💡 핵심 아이디어

- 입력 받은 자연수가 1이 될때까지 반복문을 통해 자연수가 1이 될때까지 반복하면서 반복 횟수를 더하고, 반복 횟수가 500번을 초과하면 -1을 반환한다.

---

## ✍️ 풀이 과정

1. **(데이터 입력/자료구조)**: 작업을 반복하는 횟수를 저장하는 변수 int counter
2. **(탐색/로직 1)**: 입력 받은 자연수가 1인지 확인한다.
3. **(탐색/로직 2)**: 입력 받은 자연수가 짝수라면 2로 나눈 값을 반환한다.
4. **(탐색/로직 3)**: 입력 받은 자연수가 홀수라면 3을 곱하고 1을 더한 값을 반환한다.
5. **(탐색/로직 4)**: 입력 받은 자연수가 1이 될때까지 탐색 로직을 반복하면서 반복 횟수를 더한다.
6. **(예외 처리 1)**: 주어진 수가 1이라면 0을 반환한다.
7. **(예외 처리 2)**: 반복 횟수가 500번을 초과하면 -1을 반환한다.

---

## 🧐 시간 복잡도 분석

- O(N)
- (이유: "입력 받은 자연수가 1이 될때까지 반복문을 통해 자연수가 1이 될때까지 반복하면서 반복 횟수를 더하고, 반복 횟수가 500번을 초과하면 -1을 반환하는 연산이 수행되기 때문입니다.")

---

## 🧠 오답 노트 / 배운 점

- **(실수 1)**: 초기에는 재귀 함수를 사용하는 로직을 작성했지만, 재귀 함수는 스택 오버플로우가 발생할 수 있어 반복문을 사용하는 로직으로 변경했다.
- **(실수 2)**: 자연수를 long 타입으로 변환하지 않고 int 타입으로 변환하여 범위를 초과하여 런타임 에러가 발생했다.
- **(배운 점 1)**: 재귀 함수 로직을 작성하는 방법을 배웠다.
- **(배운 점 2)**: 자연수를 long 타입으로 변환하여 범위를 초과하지 않도록 하자.
- **(느낀 점)**: 재귀 함수는 호출 스택에 따라 스택 오버플로우가 발생할 수 있어 반복문을 사용하는 로직으로 변경했다.
