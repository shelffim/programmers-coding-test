## 📝 문제 요약

- 입력 받은 배열에서 0부터 9까지의 숫자 중 없는 숫자를 모두 더한 값을 반환하는 문제

---

## 💡 핵심 아이디어

- 0부터 9까지의 숫자를 set에 저장하고, 입력 받은 배열의 요소를 제거한다.
- 남아있는 요소를 모두 더한다.
