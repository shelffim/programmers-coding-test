## 📝 문제 요약

- 단어의 가운데 글자를 반환하는 문제

---

## 💡 핵심 아이디어

- 단어의 길이가 짝수일 경우 가운데 2개의 글자를 반환하고, 홀수이면 가운데 1개의 글자를 반환한다.

---

## ✍️ 풀이 과정

1. **(데이터 입력/자료구조)**: 단어의 길이를 저장하는 변수 int s_length
2. **(탐색/로직 1)**: 입력받은 단어의 길이를 sLength에 저장한다.
3. **(탐색/로직 2)**: sLength가 짝수인 경우 가운데 2개의 글자를 반환하고, 홀수이면 가운데 1개의 글자를 반환한다.

---

## 🧐 시간 복잡도 분석

- O(1)
- (이유: "단어의 길이가 짝수인지 홀수인지 확인하고, 가운데 2개의 글자를 반환하거나 가운데 1개의 글자를 반환하는 연산이 수행되기 때문입니다.")
