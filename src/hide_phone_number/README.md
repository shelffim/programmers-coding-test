## 📝 문제 요약

- 전화번호의 뒷자리 4자리를 제외하고는 \*로 가리고, 뒷자리 4자리를 그대로 반환하는 문제

---

## 💡 핵심 아이디어

- 문자열의 길이 - 4 만큼 `*`을 저장하고, 뒷자리 4자리를 그대로 저장한다.

---

## ✍️ 풀이 과정

1. **(데이터 입력/자료구조)**: 전화번호를 저장할 변수 StringBuilder sb
2. **(탐색/로직 1)**: 입력받은 문자열의 길이 - 4 만큼 `*`을 sb에 저장한다.
3. **(탐색/로직 2)**: 입력받은 문자열의 길이 - 4 부터 문자열의 길이까지의 입력받은 문자열을 sb에 저장한다.
4. **(탐색/로직 3)**: sb를 문자열로 변환하여 반환한다.

---

## 🧐 시간 복잡도 분석

- O(N)
- (이유: "입력 받은 문자열의 길이 - 4 만큼 `*`을 저장하고, 뒷자리 4자리를 그대로 저장한다. 이를 결합하여 반환하는 연산이 수행되기 때문입니다.")
