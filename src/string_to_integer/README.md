---

## 💡 핵심 아이디어

- 입력 받은 문자열의 첫번째 문자가 부호인지 확인하고, 부호가 있으면 부호를 제외한 문자열을 정수로 변환한 후 부호를 곱하여 반환한다. 부호가 없으면 문자열을 정수로 변환하여 반환한다.
